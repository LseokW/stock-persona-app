> **[페르소나 카드 보여주기]**

각 페르소나는 서로 다른 투자 전략을 가지고 있습니다.

**1. 신중함 (겁쟁이)** — 이동평균 기반의 보수적 전략입니다. 12봉 이동평균이 상승 추세이고, 현재 가격이 이동평균 근처에 있을 때만 자본의 20%로 진입합니다.

**2. 야수의 심장** — 모멘텀 추격 전략입니다. 직전 3봉에서 수익률이 +0.5% 이상이면 자본의 80%를 투입해 올라타고, 1봉이라도 -0.3% 떨어지면 즉시 탈출합니다.

**3. 공포에 투자 (청개구리)** — 역추세 전략입니다. 직전 봉이 -1% 이상 급락하면 반등을 기대하고 진입합니다. 1% 수익 또는 1% 손실 시 청산합니다.

**4. 초보 예언가 (AI형)** — 나머지 셋이 규칙 기반이라면, 이건 LSTM 딥러닝 모델이 상승 확률을 예측해서 매매합니다. 뒤에서 더 자세히 설명하겠습니다.

### [07:00 ~ 08:30] LSTM 모델 구조 설명

> **화면: VSCode → `backend/app/ml/model.py`**

핵심은 `LSTMDirectionModel`입니다.

```python
class LSTMDirectionModel(nn.Module):
    def __init__(self, input_size=3, hidden_size=32, num_layers=1, dropout=0.2):
        self.lstm = nn.LSTM(input_size=3, hidden_size=32, batch_first=True)
        self.fc = nn.Sequential(
            nn.Linear(32, 16), nn.ReLU(), nn.Dropout(0.2), nn.Linear(16, 1)
        )
```

**입력**: 24개 봉(1시간봉 기준 하루치)의 데이터를 시퀀스로 받습니다.

**피처는 3가지**입니다. (`dataset.py`)
1. **log_return** — 수익률. 절대 가격 대신 수익률을 쓰는 이유는, 종목마다 주가 범위가 다르기 때문입니다.
2. **range_ratio** — `(고가 - 저가) / 종가`. 그 봉에서 얼마나 가격이 흔들렸는지 변동성 지표입니다.
3. **log_volume_norm** — 거래량은 최소~최대 차이가 수백만 배 나므로 log 변환 후 Z-score 정규화를 합니다.

**출력**: "다음 봉에서 가격이 오를 확률 p" (0~1 사이 값).
- p > 0.42 이면 매수
- p < 0.45 이면 매도

LSTM은 24봉 전체 시퀀스를 처리한 뒤, 마지막 hidden state만 꺼내서 FC 레이어에 넣습니다.
이 마지막 hidden state에 24봉 전체의 시간적 패턴이 압축되어 있습니다.

### [08:30 ~ 10:00] 학습 방법 설명

> **화면: VSCode → `backend/train.py` + `backend/models/training_curve.png`**

학습 스크립트는 `train.py`입니다.

```bash
uv run python train.py --ticker AAPL --epochs 50
```

**학습 과정**은 다음과 같습니다.

**1단계. 데이터 수집 & 분할**
yfinance로 2년치 1시간봉 데이터를 가져옵니다.
그 다음 **시간 순서를 유지한 채로** 70% 훈련 / 15% 검증 / 15% 테스트로 나눕니다.

중요한 점은 랜덤으로 섞지 않는다는 겁니다. 미래 데이터가 훈련에 섞이면 — look-ahead bias — 실제보다 훨씬 좋은 성능이 나옵니다. 이 함정을 피하기 위해 반드시 시간 순서 분할을 해야 합니다.

**2단계. 슬라이딩 윈도우**
24봉짜리 윈도우를 한 칸씩 이동시키며 샘플을 만듭니다.
약 600개의 훈련 샘플이 만들어집니다.

**3단계. 학습**
- 손실함수: `BCEWithLogitsLoss` — sigmoid와 이진 교차 엔트로피를 합친 것
- 옵티마이저: `Adam(lr=1e-3)`
- **Early Stopping**: 검증 손실이 5번 연속 개선되지 않으면 학습을 조기 종료합니다. 과적합을 방지하기 위해서입니다.
- 검증 정확도 기준으로 가장 좋은 모델을 저장합니다.

> **[training_curve.png 이미지 보여주기]**

이게 실제 학습 곡선입니다.
왼쪽이 train/val loss, 오른쪽이 검증 정확도입니다.
점선이 50%, 즉 랜덤 예측 기준선입니다.

최종 검증 정확도는 약 **52~54%** 수준입니다.
50%보다 겨우 높은 수준이지만, 이게 실제로 주가 예측의 현실입니다.
시장을 예측하는 건 그만큼 어렵습니다.

**4단계. 추론 (실시간)**
학습된 모델(`lstm_direction.pt`)을 `Predictor` 클래스가 로드해서, 백테스트할 때마다 24봉 윈도우를 넣어 실시간으로 상승 확률을 계산합니다.

## 예상 질문 & 답변

**Q. 왜 1시간봉을 사용했나요?**
A. 1분봉은 데이터 양이 너무 많고 노이즈가 심합니다. 일봉은 단타라고 볼 수 없습니다. 1시간봉이 단타의 패턴을 학습하기에 적절한 단위라고 판단했습니다.

**Q. 정확도가 52%인데 의미가 있나요?**
A. 주가 예측에서 50% 이상이면 이론적으로 수익이 가능합니다. 하지만 수수료와 슬리피지를 포함하면 실질적으로 무의미해집니다. 이 앱이 바로 그 현실을 보여줍니다.

**Q. 다른 모델(Transformer 등)은 써보셨나요?**
A. 훈련 샘플이 약 600개로 매우 적어서 파라미터가 많은 모델은 즉시 과적합됩니다. 의도적으로 hidden_size=32, num_layers=1의 작은 LSTM을 선택했습니다.

**Q. 실제 투자에 쓸 수 있나요?**
A. 이 앱은 교육 목적입니다. 백테스트 결과가 미래 수익을 보장하지 않으며, 실제 투자에는 수수료, 슬리피지, 세금 등이 포함됩니다.
