# AI 페르소나 주식 백테스트 웹앱

S&P 500 종목 1개를 선택하면 4가지 AI 트레이더 페르소나가 최근 6개월 데이터로 백테스트를 수행하고 수익률·위험 지표를 비교해주는 교육용 웹 앱.

## 기기별 역할

| 기기 | 역할 |
|------|------|
| 데스크탑 | 백엔드 개발 + LSTM 모델 학습 |
| 노트북 | 프론트엔드(React) 개발 |

두 기기는 GitHub를 통해 코드를 동기화한다.

## 개발 환경 설정

```bash
cd backend
uv sync
```

## 실행 방법

```bash
cd backend
uv run python try_data.py
```
